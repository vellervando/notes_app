# Praktikum 7: CRUD Notes Application

Aplikasi Catatan (*Notes App*) berbasis web dinamis yang mendukung operasi CRUD penuh (**Create, Read, Update, Delete**). Proyek ini dibuat sebagai bagian dari pemenuhan tugas Praktikum 7 untuk mendemonstrasikan manipulasi DOM, manajemen *state* data, dan implementasi arsitektur kode yang bersih (*clean code*).

🌐 **Demo Aplikasi:** [praktikum-7-crud-notes.vercel.app](https://praktikum-7-crud-notes.vercel.app/)

## 🚀 Fitur Utama
* **Create (Tambah):** Pengguna dapat membuat catatan baru dengan judul dan isi dokumen.
* **Read (Tampilkan):** Menampilkan daftar seluruh catatan secara dinamis dalam bentuk *grid/card layouts*.
* **Update (Ubah):** Mengedit isi dokumen atau judul catatan yang telah tersimpan sebelumnya tanpa mengganti entitas data secara keseluruhan.
* **Delete (Hapus):** Menghapus catatan secara permanen dari daftar penyimpanan aplikasi.
* **Desain Responsif:** Antarmuka adaptif yang optimal diakses lewat perangkat mobile, tablet, maupun desktop.
